# 📄 EasyDocs

A lightweight Flutter application for organizing and viewing your documents in one place.

---

## 📖 About

I built **EasyDocs** to solve a problem I faced myself. My phone often couldn't open PDF or DOCX files without installing a third-party viewer, and the apps I tried were filled with ads and unnecessary features.

As a developer, I wanted a simple, offline document viewer that focuses only on reading documents, with no ads, accounts, or subscriptions.

This project is also my way of learning **Flutter** and mobile app development from scratch by building something practical that I would actually use.

---

## ✨ Features

* Automatically discover supported documents on your device
* Display all documents in one place
* Search documents by name
* Open supported document types (PDF, DOCX, and more)
* Rename documents
* Delete documents

## 🛠 Tech Stack

> **Note:** The technologies listed below are tentative and may change as the project evolves. The final implementation may use different packages or libraries based on development needs.

* **Local Storage:** Hive (or SQLite)
